## Known Infrastructure Note

The `afterAll` cleanup hook (deletes test records after the run) can occasionally hit
Playwright's global 30 s test timeout when using the default `playwright.config.ts`.
The dedicated config in `config/` sets `retries: 0` and the spec uses `test.setTimeout(0)`
inside the hook to remove this limit. Always use the dedicated config (command above) to
avoid unintended retries.
