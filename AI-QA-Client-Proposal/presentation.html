Technology Stack

Tools & AI Used — Minimal Cost

Tool / AI Role Subscription
Claude Code (Anthropic AI) Reads requirements · Writes test scripts · Generates page objects & reports Paid · $20–$200+/mo
Playwright (Microsoft) Automated browser test execution — runs all UI test cases Free & Open Source
TypeScript Language for all test scripts — type-safe, maintainable Free
Node.js Runtime environment — executes Playwright test suites Free
VS Code (IDE) Development environment — optional, any editor works Free
Chromium (via Playwright) Browser for automated test execution Free · Bundled
Only 1 Paid Tool
Claude AI subscription is the only cost. All testing infrastructure (Playwright, TypeScript, Node.js) is completely free and open source.
No Vendor Lock-in
All generated scripts are plain TypeScript files. The framework runs independently of any AI subscription — tests continue to work even without Claude.

Deliverables

What You Get With Every Test Run

📁
Structured Folder Per Feature
Each requirement gets its own dedicated folder — test cases, scripts, page objects, fixtures, test data, config, and outputs all in one place.
📊
Custom HTML Dashboard
Every execution produces a standalone HTML report — donut chart, suite progress bars, test results table, acceptance criteria, environment card.
🕒
Full Run History
Re-runs are never overwritten. Each execution is stored in a timestamped folder (YYYYMMDD-HHMM). Compare any two runs side by side.
🎭
Page Object Model
AI generates maintainable Page Object files for each screen under test — making tests easy to update when the UI changes.
🖼️
Screenshots & Traces
Playwright automatically captures screenshots and full interaction traces on failures — making debugging visual and fast.
📋
Acceptance Criteria Report
Each report explicitly shows every acceptance criterion from the requirement, flagged as MET or NOT MET — ready to share with stakeholders.
